Dependencies:

Redux
### `npm i redux`
React-Redux
### `npm i react-redux`
Redux-Persist
### `npm i redux-persist`
React-Router-Dom
### `npm i --save react-router-dom`

#

To view the site, run:

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
